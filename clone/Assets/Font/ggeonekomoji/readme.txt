■解説
本ファイルは武蔵システム（http://opentype.jp）の TTEdit および　手書きでフォント　を使って作成しました。

正式名は「g_ジオメトリックねこもじ」です。
ロゴ・見出し等のディスプレイ用途を想定した、ちょっとかわいい感じの
丸っこい独特の形状のかなttfフォントです。

このフォントには有料版があり、
無料版は収録文字を減らしたお試し版的なやつで、収録文字数は54字です。
全角ひらがなの清音(+フォント名表示に必要なグリフ)のみの収録となります。

有料版(ver.G)の収録文字数は177字です。
半角アルファベット、全角ひらがな、若干の記号を収録しています。
できればご購入いただきたいですが、
無料版で文字数が足りている場合は商用利用時でも購入は必須ではありません。

　詳細はこちらを参照ください。
　　→http://material.animehack.jp/font_ggeonekomoji.html
