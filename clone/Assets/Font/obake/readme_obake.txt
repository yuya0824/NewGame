■特徴
フリーのカタカナフォントです。
現在カタカナと、「。」などの一部の記号のみのカバーです。
形は末広がりで少し血がたれたイメージです。
http://www.ankokukoubou.com/で見本がご覧いただけます。
入力方法は日本語入力でカタカナに変換してください。
（直接に半角英数から入力ではありません。）
■利用条件
フリーフォントです。無料でお使いいただけます。
画像素材、印刷物等に加工利用して配布、販売することもできます。
商用利用の場合、なるべく作者に何に使ったか簡単にメール報告をくださるか、
サイト等に使った旨を表記あるいはリンクしてくださると嬉しいです。（必須ではありません。）
■禁止事項
フォントファイルの再配布。

雑誌等に掲載をご希望の際には条件付でOKすることもあります。(切手をいくらか送ってもらうなど）
OS等に組み込む場合は、条件付でOKすることもあります。(公式サイトからのリンクなど）
